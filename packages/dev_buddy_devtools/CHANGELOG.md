# Changelog

## 0.2.0

### Added
- `DevBuddyServiceExtensions` — Flutter DevTools integration via service extensions
- `ext.dev_buddy.snapshot` — Full diagnostic snapshot
- `ext.dev_buddy.events` — Paginated event history with module filter
- `ext.dev_buddy.state` — State store contents for time-travel
- `ext.dev_buddy.clear` — Reset diagnostic state
- Safe re-registration (supports hot restart)
- Query parameters for filtering and pagination
