# Changelog

## 0.2.0

### Added
- `DevBuddyBlocObserver` — BLoC/Cubit state tracking for DevBuddy
- Records `onTransition` for BLoCs with event type tracking
- Records `onChange` for Cubits with state diff
- Records `onError` with serialized exception data
- Custom serializer support for complex state objects
- Per-source state diff computation
